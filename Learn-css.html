Getting Started with HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a webpage using markup.

Key HTML Elements

  • <h1> to <h6>: Headings
  • <p>: Paragraphs
  • <a>: Links
  • <img>: Images
  • <div>: Division or section
